# 🧮 Java Math Games

A simple yet fun terminal-based math game written entirely in Java. 
Test your mental reflexes by solving random additions, subtractions, multiplications, divisions, and modulo operations.

## ✨ Features

* **Dynamic Difficulty**: The user can choose the difficulty level, which determines the number of digits in the generated numbers (e.g., Level 1 = numbers from 1 to 9; Level 2 = numbers from 10 to 99).
* **Random Operations**: The system automatically generates 5 types of math operations (`+`, `-`, `*`, `/`, `%`).
* **Safety Checks**: Built-in logic to prevent fatal errors, such as dividing by zero.
* **Final Score**: At the end of the round, the program calculates and displays your total score.

## 🎮 How to Play

1. Start the program.
2. Enter the **number of operations** you want to tackle in the current session.
3. Choose the **difficulty level** (e.g., type `2` to calculate with two-digit numbers).
4. Type your answer and press `Enter`. Note that divisions are **integer divisions** (no decimals).
5. Check your final score at the end of the game :)
